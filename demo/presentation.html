Architecture Technique

  • Frontend: Thymeleaf + Bootstrap 5 (Glassmorphism)
  • Backend: Spring Boot 3.2 (Java 21)
  • Security: Spring Security 6 (RBAC, BCrypt)
  • Data: PostgreSQL + JPA (Hibernate)
  • AI: Spring AI + Google Gemini API
